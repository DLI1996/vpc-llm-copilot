AWS Transit Gateway is a network transit hub that allows you to connect your virtual private clouds (VPCs) and on-premises networks. It supports inter-Region peering and automatically encrypts network traffic between AWS data centers. Transit Gateway has key concepts such as attachments, transit gateway Maximum Transmission Unit (MTU), transit gateway route table, associations, and route propagation. You can create and manage transit gateways using the AWS Management Console, AWS CLI, AWS SDKs, or Query API. Pricing for transit gateway includes charges for attachments and traffic processed.

This documentation is for the AWS API reference for the DeleteTransitGatewayPrefixListReference action. This action is used to delete a reference to a prefix list in a specified transit gateway route table. The request parameters include the DryRun parameter, which checks for required permissions, and the PrefixListId and TransitGatewayRouteTableId parameters, which specify the ID of the prefix list and route table, respectively. The response includes the requestId and transitGatewayPrefixListReference elements. The documentation also provides examples and links to language-specific AWS SDKs for further information.

This documentation provides instructions on how to view and configure your Amazon Virtual Private Cloud (VPC) using the AWS console and CLI. It includes steps to view VPC details, visualize resources in your VPC, add and remove IPv4 and IPv6 CIDR blocks, and associate and disassociate CIDR blocks with your VPC. The documentation also provides information on creating and deleting a VPC, as well as additional resources for managing VPCs.

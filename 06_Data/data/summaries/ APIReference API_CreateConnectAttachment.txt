This documentation is about AWS Global Networks for Transit Gateways and specifically focuses on the API reference for the "CreateConnectAttachment" action. It provides the request syntax, URI request parameters, request body, response syntax, response elements, and possible errors. The "CreateConnectAttachment" action is used to create a core network Connect attachment, which is a GRE-based tunnel attachment that establishes a connection between a core network and an appliance using an existing VPC attachment as the transport mechanism.

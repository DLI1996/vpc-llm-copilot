This documentation is about the AWS API action "UnassignPrivateNatGatewayAddress" which is used to unassign secondary private IPv4 addresses from a private NAT gateway. The documentation provides information about the request parameters, response elements, errors, and related resources. It also mentions that during the unassigning process, additional IP addresses cannot be assigned or unassigned while the connections are being drained. The documentation also specifies that a private IP address will only be released at the end of the specified maximum drain duration.

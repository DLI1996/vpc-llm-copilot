This documentation is about the AWS Global Networks for Transit Gateways API, specifically the GetCoreNetworkChangeEvents operation. It provides information about a core network change event. The request requires the coreNetworkId, maxResults, and policyVersionId parameters. The response includes an array of CoreNetworkChangeEvent objects and a nextToken for pagination. Possible errors include AccessDeniedException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources for using the API in different programming languages.

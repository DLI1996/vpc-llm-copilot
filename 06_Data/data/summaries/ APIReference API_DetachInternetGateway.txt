This is a summary of the documentation for the AWS API action "DetachInternetGateway". This action is used to detach an internet gateway from a VPC, disabling connectivity between the internet and the VPC. The request parameters include "DryRun" (a boolean value to check permissions), "InternetGatewayId" (the ID of the internet gateway), and "VpcId" (the ID of the VPC). The response elements include "requestId" (the ID of the request) and "return" (a boolean value indicating success or error). The documentation also provides examples and references to other related APIs and SDKs.

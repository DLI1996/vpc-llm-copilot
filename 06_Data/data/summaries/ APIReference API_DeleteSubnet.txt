The AWS API Reference provides documentation for the DeleteSubnet API action in Amazon EC2. This action is used to delete a specified subnet, but all running instances in the subnet must be terminated before the subnet can be deleted. The request parameters for this action include the DryRun parameter, which checks for required permissions, and the SubnetId parameter, which specifies the ID of the subnet to be deleted. The response elements include the requestId and return values. The documentation also provides examples of how to use this API action and references to other related APIs and SDKs.

This document provides a user guide for enabling internet access in an Amazon Virtual Private Cloud (VPC) using an internet gateway. It explains the steps to create a subnet, attach an internet gateway, create a custom route table, create a security group for internet access, assign an Elastic IP address to an instance, detach an internet gateway from a VPC, and delete an internet gateway. It also provides information on pricing and includes API and command overviews for performing these tasks programmatically.

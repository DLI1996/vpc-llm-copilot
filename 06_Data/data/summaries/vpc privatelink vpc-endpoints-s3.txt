This document provides information about using gateway endpoints to access Amazon S3 from your VPC. It explains how to create a gateway endpoint, control access using bucket policies, associate route tables, edit the VPC endpoint policy, and delete a gateway endpoint. It also discusses considerations and best practices for using gateway endpoints, such as configuring private DNS and controlling access to specific resources. The document includes step-by-step instructions for creating and managing gateway endpoints using the AWS Management Console and command line interface.

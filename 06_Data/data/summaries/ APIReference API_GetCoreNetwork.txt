This documentation is about the AWS Global Networks for Transit Gateways API, specifically the GetCoreNetwork operation. It provides information about the LIVE policy for a core network. The request requires the coreNetworkId parameter and does not have a request body. The response includes details about the core network in JSON format. Possible errors include AccessDeniedException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources and language-specific AWS SDKs.

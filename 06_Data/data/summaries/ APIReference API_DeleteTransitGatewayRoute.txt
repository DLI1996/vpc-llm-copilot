This documentation is about the AWS API reference for the DeleteTransitGatewayRoute action. It provides information about the request parameters, response elements, errors, and related resources. The DeleteTransitGatewayRoute action is used to delete a specific route from a transit gateway route table. The required parameters for this action are DestinationCidrBlock and TransitGatewayRouteTableId. The response includes the requestId and route information. The documentation also provides links to language-specific AWS SDKs and information on how to enable JavaScript for using the Amazon Web Services Documentation.

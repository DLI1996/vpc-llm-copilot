This documentation is about the AWS Global Networks for Transit Gateways API. It provides information on how to request network resource relationships for a specified global network. The API supports various URI request parameters such as AccountId, AwsRegion, CoreNetworkId, globalNetworkId, MaxResults, NextToken, RegisteredGatewayArn, ResourceArn, and ResourceType. The response includes NextToken and Relationships. The documentation also includes information on possible errors and provides links to related resources and SDKs.

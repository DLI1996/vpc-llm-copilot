This documentation is about the AWS Global Networks for Transit Gateways API, specifically the GetTransitGatewayConnectPeerAssociations operation. It provides information about one or more transit gateway Connect peer associations in a global network. The request syntax includes URI request parameters such as globalNetworkId, MaxResults, NextToken, and TransitGatewayConnectPeerArns. The response includes NextToken and an array of TransitGatewayConnectPeerAssociation objects. The documentation also lists possible errors and provides links to additional resources for using the API in different programming languages.

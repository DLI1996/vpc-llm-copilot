This documentation is for the AWS API action "EnableTransitGatewayRouteTablePropagation". It enables a specified attachment to propagate routes to a specified propagation route table. The request parameters include "DryRun" (a boolean value to check permissions), "TransitGatewayAttachmentId" (the ID of the attachment), "TransitGatewayRouteTableAnnouncementId" (the ID of the transit gateway route table announcement), and "TransitGatewayRouteTableId" (the ID of the propagation route table). The response includes information about route propagation and the ID of the request. Common errors and language-specific SDKs are also mentioned.

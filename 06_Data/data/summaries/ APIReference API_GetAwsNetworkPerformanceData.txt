This documentation is for the AWS API reference for the GetAwsNetworkPerformanceData action. It provides information about the request parameters, response elements, errors, and related resources. The GetAwsNetworkPerformanceData action is used to retrieve network performance data. The request parameters include DataQuery, DryRun, EndTime, MaxResults, NextToken, and StartTime. The response elements include dataResponseSet, nextToken, and requestId. The documentation also provides links to language-specific AWS SDKs and information on how to enable JavaScript for using the Amazon Web Services Documentation.

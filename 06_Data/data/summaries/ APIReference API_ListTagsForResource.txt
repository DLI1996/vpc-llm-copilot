This documentation is for the AWS Global Networks for Transit Gateways API. It provides information on how to list the tags for a specified resource. The request requires the resource's Amazon Resource Name (ARN) as a parameter. The response includes a list of tags associated with the resource. The API may return errors such as access denied, internal server error, resource not found, throttling, or validation failure. The documentation also provides links to language-specific AWS SDKs for using the API.

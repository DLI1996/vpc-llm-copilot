AWS Global Networks for Transit Gateways is a service that allows users to create and manage global networks across AWS accounts, regions, and on-premises locations. It provides a central dashboard for managing network objects such as transit gateways, devices, connections, links, and sites. The home region for global networks is currently limited to US West (Oregon). The service is available in multiple AWS regions and there are no additional fees for using it, only standard fees for managing network resources. Users can refer to the AWS Global Networks for Transit Gateways User Guide for more information on how to get started and pricing details.

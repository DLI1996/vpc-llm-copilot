This documentation is about the AWS API operation "DeleteTargetGroup", which is used to delete a target group. The request requires the target group identifier as a parameter and does not have a request body. The response includes the ARN, ID, and status of the target group. Possible errors include ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources for using this API in different programming languages.

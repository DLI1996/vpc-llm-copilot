This section of the AWS documentation explains how to view the details of a resource discovery in Amazon VPC IP Address Manager (IPAM). It provides instructions on how to access the IPAM console and navigate to the resource discovery details. The details include information about the discovered resources, such as VPCs, subnets, and IP addresses, as well as the discovered accounts being monitored. The documentation also mentions the commands that can be used with the AWS CLI to view resource discovery details.

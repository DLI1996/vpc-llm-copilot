This documentation is about the AWS API reference for the CreateTransitGatewayConnectPeer action. It explains the request parameters, response elements, and errors associated with this action. The CreateTransitGatewayConnectPeer action is used to create a Connect peer for a specified transit gateway Connect attachment between a transit gateway and an appliance. The documentation also provides information about the required and optional parameters for this action, as well as links to additional resources and SDKs for using this API.

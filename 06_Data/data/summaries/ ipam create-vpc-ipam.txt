This document provides instructions on how to create a Virtual Private Cloud (VPC) in Amazon Web Services (AWS) using an IP Address Manager (IPAM) pool CIDR. It explains that when creating a VPC, users have the option to choose a CIDR from an IPAM pool, and AWS will allocate a CIDR from the pool. The document also mentions that complete instructions using the AWS CLI can be found in the Tutorials for Amazon VPC IP Address Manager section. Additionally, it states that Javascript must be enabled to access the AWS Documentation.

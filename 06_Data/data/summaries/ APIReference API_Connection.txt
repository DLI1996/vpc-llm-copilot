This is a summary of the documentation for AWS Global Networks for Transit Gateways. It provides information about the Connection object, which describes a connection. The Connection object includes various attributes such as ConnectedDeviceId, ConnectedLinkId, ConnectionArn, ConnectionId, CreatedAt, Description, DeviceId, GlobalNetworkId, LinkId, State, and Tags. The documentation also provides links to language-specific AWS SDKs for more information on using the API.

This documentation is for the AWS API action "DisassociateTransitGatewayRouteTable". It provides information about the request parameters, response elements, errors, and related resources. The action is used to disassociate a resource attachment from a transit gateway route table. The required parameters for this action are "TransitGatewayAttachmentId" and "TransitGatewayRouteTableId". The response includes information about the association and a request ID. The documentation also provides links to additional resources for using this API in different programming languages.

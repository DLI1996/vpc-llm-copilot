AWS services support access through public endpoints. Some AWS services also support access using private endpoints powered by AWS PrivateLink. AWS services can support IPv6 through their private endpoints even if they do not support IPv6 through their public endpoints. Endpoints that support IPv6 can respond to DNS queries with AAAA records.Services that support IPv6The following table lists the AWS services that provide dual stack support, IPv6 only support, and endpoints that support IPv6.

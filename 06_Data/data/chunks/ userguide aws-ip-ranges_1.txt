To view the current ranges, download the .json file. To maintain history, save successive versions of the .json file on your system. To determine whether there have been changes since the last time that you saved the file, check the publication time in the current file and compare it to the publication time in the last file that you saved.The IP address ranges that you bring to AWS through bring your own IP addresses (BYOIP) are not included in the .json file.Alternatively, some services publish their address ranges using AWS-managed prefix lists.

For Location type, specify whether the device is located in a remote location (On-premises, Data center/ Other Cloud Provider) or in the AWS Cloud. If you choose AWS Cloud, specify the location of the device within AWS: For the Zone, specify the name of an Availability Zone,Local Zone, Wavelength Zone, or an Outpost. For the Subnet, specify the Amazon Resource Name (ARN) of the subnet (for example, arn:aws:ec2:useast-1:111111111111:subnet/subnet-abcd1234).For Address, enter the physical location of the site (for example New York, NY 10004).For Latitude, enter the latitude coordinates for the site (for example, 40.7128).

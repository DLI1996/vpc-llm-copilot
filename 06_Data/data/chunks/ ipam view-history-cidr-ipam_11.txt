Overlap status: The overlap status of CIDR. Nonoverlapping: The resource CIDR does not overlap with another CIDR in the same scope. Overlapping: The resource CIDR overlaps with another CIDR in the same scope. Note that if a resource CIDR is overlapping, it could be overlapping with a manual allocation. Ignored: The managed resource has been chosen to be exempt from monitoring. IPAM does not evaluate ignored resources for overlap or allocation rule compliance.

To create a static route or blackhole route using the AWS CLIUse the create-transit-gateway-route command.Delete a static routeYou can delete static routes from a transit gateway route table.To delete a static route using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.On the navigation pane, choose Transit Gateway Route Tables.Select the route table for which to delete the route, and choose Routes.Choose the route to delete.Choose Delete static route.In the confirmation box, choose Delete static route.To delete a static route using the AWS CLIUse the delete-transit-gateway-route command.Replace a static routeYou can replace a static route in a transit gateway route table with a different static route.To replace a static route using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.

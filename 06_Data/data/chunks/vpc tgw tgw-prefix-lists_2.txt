For Transit gateway attachment ID, choose the ID of the attachment to which to route traffic.Choose Create prefix list reference.To create a prefix list reference using the AWS CLIUse the create-transit-gateway-prefix-list-reference command.View prefix list referencesYou can view the prefix list references in your transit gateway route table. You can also view each entry in the prefix list as an individual route in your transit gateway route table. The route type for a prefix list route is propagated.To view a prefix list reference using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Transit Gateway Route Tables.Select the transit gateway route table.In the lower pane, choose Prefix list references. The prefix list references are listed.

AWSDocumentationAWS Network ManagerInfrastructure Performance User GuideHow latency is calculatedRegion availabilityHow Infrastructure Performance worksNetwork latency measurements are an aggregate of latency performance information captured by active probing between AWS managed probes within the AWS global network.Inter-Region latency metrics are generated by aggregating latency measurements from probes located across AWS Regions, filtered on your chosen source and destination AWS Region pairs.Inter-Availability Zone latency metrics are generated by aggregating latency measurements from probes located across Availability Zones, filtered on your chosen source and destination Availability Zones.

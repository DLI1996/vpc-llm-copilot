(Optional) To create a dual stack VPC, specify an IPv6 address range for your VPC. For IPv6 CIDR block, do one of the following: Choose IPAM-allocated IPv6 CIDR block and select your IPAM IPv6 address pool. The size of the CIDR block is limited by the allocation rules on the IPAM pool. Choose Amazon-provided IPv6 CIDR block to request an IPv6 CIDR block from an Amazon pool of IPv6 addresses. For Network Border Group, select the group from which AWS advertises IP addresses. Amazon provides a fixed IPv6 CIDR block size of /56. Choose IPv6 CIDR owned by me to use an IPv6 CIDR block that you brought to AWS using bring your own IP addresses (BYOIP). For Pool, choose the IPv6 address pool from which to allocate the IPv6 CIDR block.

AWSDocumentationAmazon EC2API ReferenceRequest ParametersResponse ElementsErrorsExamplesSee AlsoAssociateRouteTableAssociates a subnet in your VPC or an internet gateway or virtual private gateway attached to your VPC with a route table in your VPC. This association causes traffic from the subnet or gateway to be routed according to the routes in the route table. The action returns an association ID, which you need in order to disassociate the route table later. A route table can be associated with multiple subnets.For more information, see Route tables in the Amazon VPC User Guide.Request ParametersThe following parameters are for this specific action.

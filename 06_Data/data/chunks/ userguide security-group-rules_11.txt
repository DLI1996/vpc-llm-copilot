For each rule, choose Add rule and do the following. For Type, choose the type of protocol to allow. For TCP or UDP, you must enter the port range to allow. For custom ICMP, you must choose the ICMP type name from Protocol, and, if applicable, the code name from Port range. For any other type, the protocol and port range are configured automatically. For Source type (inbound rules) or Destination type (outbound rules), do one of the following to allow traffic: Choose Custom and then enter an IP address in CIDR notation, a CIDR block, another security group, or a prefix list. Choose Anywhere-IPv4 to allow traffic from any IPv4

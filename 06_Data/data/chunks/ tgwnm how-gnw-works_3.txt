You can create a global network that includes transit gateways in multiple AWS Regions and accounts. This enables you to monitor the global health of your AWS network. In the following diagram, the global network includes a transit gateway in the us-east-2 Region from Account A and a transit gateway in the us-west-2 Region from Account B. Each transit gateway has VPC and VPN attachments.

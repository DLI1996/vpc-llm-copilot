Create an interface endpointUse the following procedure to create an interface VPC endpoint that connects to the SaaS product.RequirementSubscribe to the service.To create an interface endpoint to a partner serviceOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoints.Choose Create endpoint.If you purchased the service from AWS Marketplace, do the following: For Service category, choose AWS Marketplace services. Enter the name of the service.If you subscribed to a service with the AWS Service Ready designation, do the following: For Service category, choose PrivateLink Ready partner services. Enter the name of the service and choose Verify service.For VPC, select the VPC from which you'll access the product.For Subnets, select one subnet per Availability Zone from which you'll access the product.

Infrastructure Performance shares its API namespace with Amazon EC2. Policy actions in Infrastructure Performance use the following prefix before the action: ec2:. For example, to grant someone permission to create a path with the GetAwsNetworkPerformanceData API operation, you include the ec2:GetAwsNetworkPerformanceData action in their policy. Policy statements must include either an Action or NotAction element.To specify multiple actions in a single statement, separate them with commas as shown in the following example.You can specify multiple actions using wildcards (*).

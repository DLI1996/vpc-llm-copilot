For each rule, inbound or outbound, choose Add rule, and then specify the following information: Number: The rule priority. Rule action: Indicates whether to accept or reject the packets. Protocol: The protocol. (Optional) Source port range: The source port range. (Optional) Destination port range: The destination port range. Source CIDR block: The source CIDR block. The source and destination CIDR blocks must both be either IPv4 ranges or IPv6 ranges. Destination CIDR block: The destination CIDR block. The source and destination CIDR blocks must both be either IPv4 ranges or IPv6 ranges.

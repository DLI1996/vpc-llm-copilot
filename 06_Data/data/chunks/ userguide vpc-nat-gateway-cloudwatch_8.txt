PacketsDropCount The number of packets dropped by the NAT gateway. A value greater than 0.01 percent might indicate an ongoing transient issue with the NAT gateway. If this value exceeds 0.01 percent of the total traffic on the NAT gateway, check the AWS service health dashboard . Units: Count Statistics: The most useful statistic is Sum . PacketsInFromDestination The number of packets received by the NAT gateway from the destination.

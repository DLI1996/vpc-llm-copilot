To resize your instance, you must stop it. Stopping and starting an instance changes the public IPv4 address for the instance, if it has one. If you have any data stored on instance store volumes, the data is erased.To resize your instanceOpen the Amazon EC2 console at https://console.aws.amazon.com/ec2/.In the navigation pane, choose Instances, and select the database instance.Choose Actions, Instance State, Stop.In the confirmation dialog box, choose Yes, Stop.With the instance still selected, choose Actions, Instance Settings, Change Instance Type.For Instance Type, choose the new instance type, and then choose Apply.

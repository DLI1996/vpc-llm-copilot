Select the check box for an attachment where the Resource Type is Transit gateway route table. Details about the attachment are displayed in the lower part of the page. In this section, you can also edit the attachment Tags by choosing the Tags tab.In the details section, choose the Tags tab.Choose Add/Update tags.In the Tags section, add Key and Value tags to help identify this transit gateway route table. You can add multiple tags by choosing Add tag, or remove any tag by choosing Remove tag.When finished, choose Edit attachment.If you made any changes, choose Edit attachment to save the changes. The Attachments page displays along with a confirmation that the attachment was modified successfully.

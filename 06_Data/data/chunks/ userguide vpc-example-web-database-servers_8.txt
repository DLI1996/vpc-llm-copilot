On the dashboard, choose Create VPC.For Resources to create, choose VPC and more.Configure the VPC For Name tag auto-generation, enter a name for the VPC. For IPv4 CIDR block, you can keep the default suggestion, or alternatively you can enter the CIDR block required by your application or network. For more information, see VPC CIDR blocks. (Optional) If your application communicates by using IPv6 addresses, choose IPv6 CIDR block, Amazon-provided IPv6 CIDR block.

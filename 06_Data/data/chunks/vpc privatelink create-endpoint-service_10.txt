Open the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoints.Choose Create endpoint.For Service category, choose Other endpoint services.For Service name, enter the name of the service (for example, com.amazonaws.vpce.us-east-1.vpce-svc-0e123abc123198abc), and choose Verify service.For VPC, select a VPC in which to create the endpoint.For Subnets, select the subnets (Availability Zones) from which you'll access the endpoint service.

Select the check box for an attachment where the Resource Type is Connect. Details about the attachment are displayed, as well as any Connect peers and tags that are associated with the attachment. You can add new Connect peers and add, remove, or edit tags. To add a new Connect peer attachment, see Add a Connect peer. To add or edit attachment Tags, choose the Tags tab. The current list of tags associated with this attachment are displayed. Choose Edit tags to modify or delete current tags, and to add new tags.Choose Edit.

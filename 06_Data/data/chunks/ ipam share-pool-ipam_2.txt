To share an IPAM pool using RAMOpen the IPAM console at https://console.aws.amazon.com/ipam/. In the navigation pane, choose Pools. By default, the default private scope is selected. If you donâ€™t want to use the default private scope, from the dropdown menu at the top of the content pane, choose the scope you want to use. For more information about scopes, see How IPAM works. In the content pane, choose the pool you want to share and choose Actions > View details. Under Resource sharing, choose Create resource share. As a result, the AWS RAM console opens. You'll create the shared pool in AWS RAM. Choose Create a resource share. Add a Name for the shared resource.

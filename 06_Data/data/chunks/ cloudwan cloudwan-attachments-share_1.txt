On the Global networks page, choose the global network ID.In the navigation pane, under Shared by me, choose Attachments.Choose Create attachment.Enter a name to identify the attachment.From the Core network dropdown list, choose the core network that is shared with you and that is where you want to create the VPC attachment.From the Edge location dropdown list, choose the location where the attachment is located.From the Attachment type dropdown list, choose VPC.In the VPC attachment section, choose IPv6 support if you want to enable IPv6 for the attachment.

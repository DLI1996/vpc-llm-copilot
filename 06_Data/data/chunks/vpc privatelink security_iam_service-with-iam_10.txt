For more information, see Actions Defined by Amazon EC2 in the Service Authorization Reference.Policy resources for AWS PrivateLinkSupports policy resources YesAdministrators can use AWS JSON policies to specify who has access to what. That is, which principal can perform actions on what resources, and under what conditions.The Resource JSON policy element specifies the object or objects to which the action applies. Statements must include either a Resource or a NotResource element. As a best practice, specify a resource using its Amazon Resource Name (ARN). You can do this for actions that support a specific resource type, known as resource-level permissions.

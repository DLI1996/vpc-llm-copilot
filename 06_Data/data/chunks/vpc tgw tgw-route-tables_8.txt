On the navigation pane, choose Transit Gateway Route Tables.Choose the route that you want to replace in the route table.In the details section, choose the Routes tab.Choose Actions, Replace static route.For the Type, choose either Active or Blackhole.From the Choose attachment drop-down, choose the transit gateway that will replace the current one in the route table.Choose Replace static route.To replace a static route using the AWS CLIUse the replace-transit-gateway-route command.Export route tables to Amazon S3You can export the routes in your transit gateway route tables to an Amazon S3 bucket. The routes are saved to the specified Amazon S3 bucket in a JSON file.To export transit gateway route tables using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.

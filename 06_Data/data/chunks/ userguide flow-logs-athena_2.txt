Generate the CloudFormation template using the consoleAfter the first flow logs are delivered to your S3 bucket, you can integrate with Athena by generating a CloudFormation template and using the template to create a stack.RequirementsThe selected Region must support AWS Lambda and Amazon Athena.The Amazon S3 buckets must be in the selected Region.The log record format for the flow log must include the fields used by the specific predefined queries that you'd like to run.To generate the template using the console

To view a multicast domain using the console Open the Amazon VPC console at https://console.aws.amazon.com/vpc/. On the navigation pane, choose Transit Gateway Multicast. Select the multicast domain. Choose the Associations tab.To view a multicast domain using the AWS CLI Use the describe-transit-gateway-multicast-domains command.Disassociating subnets from a multicast domainUse the following procedure to disassociate subnets from a multicast domain.To disassociate subnets using the console Open the Amazon VPC console at https://console.aws.amazon.com/vpc/. On the navigation pane, choose Transit Gateway Multicast. Select the multicast domain. Choose the Associations tab. Select the subnet, and then choose Actions, Delete association.

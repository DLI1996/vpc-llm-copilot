To create a pool Open the IPAM console at https://console.aws.amazon.com/ipam/. In the navigation pane, choose Pools. Choose Create pool. Under IPAM scope, choose the public scope. For more information about scopes, see How IPAM works. By default, when you create a pool, the default private scope is selected. Pools in the private scope must be IPv4 pools. Pools in the public scope can be IPv4 or IPv6 pools. The public scope is intended for all space that can be or is currently advertised by AWS to the internet. (Optional) Add a Name tag for the pool and a description for the pool. Under Source type, choose IPAM

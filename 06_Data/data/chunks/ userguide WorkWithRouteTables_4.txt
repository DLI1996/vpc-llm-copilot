Create a custom route tableYou can create a custom route table for your VPC using the Amazon VPC console.To create a custom route table using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Route tables.Choose Create route table.(Optional) For Name, enter a name for your route table.For VPC, choose your VPC.(Optional) To add a tag, choose Add new tag and enter the tag key and tag value.Choose Create route table.To create a custom route table using the command linecreate-route-table (AWS CLI)New-EC2RouteTable (AWS Tools for Windows PowerShell)Add and remove routes from a route tableYou can add, delete, and modify routes in your route tables. You can only modify routes that you've added.

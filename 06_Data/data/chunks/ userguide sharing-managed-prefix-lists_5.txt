Open the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Managed Prefix Lists.Select the prefix list and choose Associations in the lower pane.The IDs of the resources that are referencing the prefix list are listed in the Resource ID column. The owners of the resources are listed in the Resource Owner column.To identify references to a shared prefix list using the AWS CLIUse the get-managed-prefix-list-associations command.Unshare a shared prefix list

Step 1: Create and analyze a pathSpecify the path for the traffic from a source to a destination. After you create the path, Reachability Analyzer analyzes the path once. You can analyze a path at any time to determine whether your intended connectivity is supported, even as your network configuration changes.To create a pathOpen the Network Manager console athttps://console.aws.amazon.combetworkmanager/home.In the navigation pane, choose Reachability Analyzer.Choose Create and analyze path.(Optional) For Name tag, enter a descriptive name for the analysis.

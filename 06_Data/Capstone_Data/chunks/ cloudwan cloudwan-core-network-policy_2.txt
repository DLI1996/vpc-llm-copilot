(Optional) Under Core network policy settings, set the beginning and ending ASN range (Autonomous System Number). Format the range as xxxxx - xxxxx. ASN is the Border Gateway Protocol (BGP) for the new core network. Valid ranges are 64512 – 65334 and 4200000000 – 4294967294. The ASN range is left-closed and right-open. This means that the leftmost number is included in the range but the rightmost number is not. For example, if you choose an ASN range of 64900-64903, the actual available ASN range is 64900 through 64902. 64903 is not included.

For more information, see Amazon EC2 service quotas in the AWS General Reference.Create a VPC for the NAT instanceUse the following procedure to create a VPC with a public subnet and a private subnet.To create the VPCOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.Choose Create VPC.For Resources to create, choose VPC and more.For Name tag auto-generation, enter a name for the VPC.To configure the subnets, do the following: For Number of Availability Zones, choose 1 or 2, depending on your needs. For Number of public subnets, ensure that you have one public subnet per Availability Zone. For Number of private subnets, ensure that you have one private subnet per Availability Zone.Choose Create VPC.Create a security group for the NAT instance

AWSDocumentationAmazon EC2API ReferenceRequest ParametersResponse ElementsErrorsSee AlsoCreateTrafficMirrorFilterCreates a Traffic Mirror filter.A Traffic Mirror filter is a set of rules that defines the traffic to mirror.By default, no traffic is mirrored. To mirror traffic, use CreateTrafficMirrorFilterRule to add Traffic Mirror rules to the filter. The rules you add define what traffic gets mirrored. You can also use ModifyTrafficMirrorFilterNetworkServices to mirror supported network services.Request ParametersThe following parameters are for this specific action.

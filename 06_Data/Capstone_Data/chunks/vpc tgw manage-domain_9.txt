To add tags to a multicast domain using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/. On the navigation pane, choose Transit Gateway Multicast. Select the multicast domain. Choose Actions, Manage tags. For each tag, choose Add new tag and enter a Key and Value for the tag.

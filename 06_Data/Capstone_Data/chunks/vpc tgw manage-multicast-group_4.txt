To deregister members using the console Open the Amazon VPC console at https://console.aws.amazon.com/vpc/. On the navigation pane, choose Transit Gateway Multicast. Select the multicast domain. Choose the Groups tab. Select the members, and then choose Remove member.To deregister members using the AWS CLI Use the deregister-transit-gateway-multicast-group-members command.Viewing your multicast groupsYou can view information about your multicast groups to verify that members were discovered using the IGMPv2 protocol. Member type (in the console), or MemberType (in the AWS CLI) displays IGMP when AWS discovered members with the protocol.

Do the following on the instance to enable IP forwarding such that it persists after reboot: Using a text editor, such as nano or vim, create the following configuration file: /etc/sysctl.d/custom-ip-forwarding.conf. Add the following line to the configuration file. net.ipv4.ip_forward=1 Save the configuration file and exit the text editor. Run the following command to apply the configuration file. sudo sysctl -p /etc/sysctl.d/custom-ip-forwarding.conf

After you add a record, you can check the status of the domain verification process using the Amazon VPC console. In the navigation pane, choose Endpoint services. Select the endpoint service and check the value of Domain verification status in the Details tab. If domain verification is pending, wait a few minutes and refresh the screen. If needed, you can initiate the verification process manually. Choose Actions, Verify domain ownership for private DNS name.The private DNS name is ready for use by service consumers when the verification status is verified. If the verification status changes, new connection requests are denied but existing connections are not affected.If the verification status is failed, see Troubleshoot domain verification issues.Get the name and value

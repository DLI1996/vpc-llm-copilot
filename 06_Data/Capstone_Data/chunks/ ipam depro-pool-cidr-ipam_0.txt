AWSDocumentationAmazon VPCIP Address ManagerDeprovision CIDRs from a poolFollow the steps in this section to deprovision CIDRs from an IPAM pool. When you deprovision all pool CIDRs, the pool can no longer be used for allocations. You must first provision a new CIDR to the pool before you can use the pool for allocations.ImportantYou cannot deprovision the CIDR if there are allocations in the pool. To remove allocations, see Release an allocation.

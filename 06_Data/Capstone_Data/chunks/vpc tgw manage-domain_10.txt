To delete a multicast domain using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/. On the navigation pane, choose Transit Gateway Multicast. Select the multicast domain, and then choose Actions, Delete multicast domain. When prompted for confirmation, enter delete and then choose Delete.To delete a multicast domain using the AWS CLI Use the delete-transit-gateway-multicast-domain command.Javascript is disabled or is unavailable in your browser.To use the Amazon Web Services Documentation, Javascript must be enabled.

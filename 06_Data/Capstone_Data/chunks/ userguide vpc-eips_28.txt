If you no longer need an Elastic IP address, we recommend that you release it. You incur charges for any Elastic IP address that's allocated for use with a VPC but that's not associated with an instance. The Elastic IP address must not be associated with an instance or network interface.To release an Elastic IP addressOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Elastic IPs.Select the Elastic IP address, and then choose Actions, Release Elastic IP addresses.When prompted, choose Release.Recover an Elastic IP address

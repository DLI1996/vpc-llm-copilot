If you already have a default VPC in the Region, you cannot create another one.To create a default VPC using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Your VPCs.Choose Actions, Create Default VPC.Choose Create. Close the confirmation screen.To create a default VPC using the command lineYou can use the create-default-vpc AWS CLI command.

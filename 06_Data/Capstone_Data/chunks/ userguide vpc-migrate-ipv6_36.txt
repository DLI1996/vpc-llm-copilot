Use the following procedures to configure IPv6 on Windows Server 2003 and Windows Server 2008 SP2.To ensure that IPv6 is preferred over IPv4, download the fix named Prefer IPv6 over IPv4 in prefix policies from the following Microsoft support page: https://support.microsoft.com/en-us/help/929852/how-to-disable-ipv6-or-its-components-in-windows.To enable and configure IPv6 on Windows Server 2003Get the IPv6 address of your instance by using the describe-instances AWS CLI command, or by checking the IPv6 IPs field for the instance in the Amazon EC2 console.Connect to your instance using the instance's public IPv4 address.From within your instance, choose Start, Control Panel, Network Connections, Local Area Connection.Choose Properties, and then choose Install.

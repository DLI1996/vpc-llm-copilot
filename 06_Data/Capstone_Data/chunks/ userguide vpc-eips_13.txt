To associate an Elastic IP address with an instance or network interfaceOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Elastic IPs.Select an Elastic IP address that's allocated for use with a VPC (the Scope column has a value of vpc), and then choose Actions, Associate Elastic IP address.Choose Instance or Network interface, and then select either the instance or network interface ID. Select the private IP address with which to associate the Elastic IP address.

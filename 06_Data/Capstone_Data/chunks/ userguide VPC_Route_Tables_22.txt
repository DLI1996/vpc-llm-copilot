The route table contains existing routes to CIDR blocks outside of the ranges in your VPC.Route propagation is enabled for the route table.In addition, the following rules and considerations apply:You cannot add routes to any CIDR blocks outside of the ranges in your VPC, including ranges larger than the individual VPC CIDR blocks.You can only specify local, a Gateway Load Balancer endpoint, or a network interface as a target. You cannot specify any other types of targets, including individual host IP addresses. For more information, see Example routing options.You cannot specify a prefix list as a destination.

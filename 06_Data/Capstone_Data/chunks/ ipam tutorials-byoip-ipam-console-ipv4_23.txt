Step 5: Create a public IPv4 poolCreating a public IPv4 pool is a required step for bringing a public IPv4 address to AWS to be managed with IPAM. This step should be done by the member account that will provision an Elastic IP address.This step must be done by the member account using the AWS CLI.ImportantPublic IPv4 pools and IPAM pools are managed by distinct resources in AWS. Public IPv4 pools are single account resources that enable you to convert your publicly-owned CIDRs to Elastic IP addresses. IPAM pools can be used to allocate your public space to public IPv4 pools.To create a public IPv4 pool using the AWS CLI

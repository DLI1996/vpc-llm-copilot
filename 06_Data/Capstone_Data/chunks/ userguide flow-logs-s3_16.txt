For Log file format, specify the format for the log file. Text – Plain text. This is the default format. Parquet – Apache Parquet is a columnar data format. Queries on data in Parquet format are 10 to 100 times faster compared to queries on data in plain text. Data in Parquet format with Gzip compression takes 20 percent less storage space than plain text with Gzip compression.(Optional) To use Hive-compatible S3 prefixes, choose Hive-compatible S3 prefix, Enable.(Optional) To partition your flow logs per hour, choose Every 1 hour (60 mins).

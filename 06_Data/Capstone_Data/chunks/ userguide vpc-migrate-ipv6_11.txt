Choose Add IPv6 CIDR, choose one of the following options, and then choose Select CIDR: Amazon-provided IPv6 CIDR block: Requests an IPv6 CIDR block from Amazon's pool of IPv6 addresses. For Network Border Group, select the group from which AWS advertises IP addresses. IPv6 CIDR owned by me: (BYOIP) Allocates an IPv6 CIDR block from your IPv6 address pool. For Pool, choose the IPv6 address pool from which to allocate the IPv6 CIDR block.To associate an IPv6 CIDR block with a subnetOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Subnets.

Choose the Devices tab. This page displays information about the devices that are connected to the site. If you don't see a device listed, you'll need to add it. For more information on adding devices, see Add a device.Choose the Links tab. This page displays the links that represent a connection from a device. If you don't see a link listed, you'll need to create the link. For the steps to create a link, see Add a link.Choose the VPNs tab. This page displays site-related VPN information.Choose the Monitoring tab. This page displays Data In and Data Out information for your links.From the dropdown list, choose the link that you want to view information for.

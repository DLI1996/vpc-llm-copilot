AWSDocumentationAmazon VPC LatticeAPI ReferenceContentsSee AlsoTargetDescribes a target.ContentsidThe ID of the target. If the target group type is INSTANCE, this is an instance ID. If the target group type is IP, this is an IP address. If the target group type is LAMBDA, this is the ARN of a Lambda function. If the target group type is ALB, this is the ARN of an Application Load Balancer. Type: String Length Constraints: Minimum length of 1. Maximum length of 200. Required: YesportThe port on which the target is listening. For HTTP, the default is 80. For HTTPS, the default is 443. Type: Integer Valid Range: Minimum value of 1. Maximum value of 65535.

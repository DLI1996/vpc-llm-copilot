AWSDocumentationAmazon VPCUser GuideVPC basicsVirtual private clouds (VPC)A virtual private cloud (VPC) is a virtual network dedicated to your AWS account. It is logically isolated from other virtual networks in the AWS Cloud. You can launch AWS resources, such as Amazon EC2 instances, into your VPC.Your account contains a default VPC for each AWS Region.

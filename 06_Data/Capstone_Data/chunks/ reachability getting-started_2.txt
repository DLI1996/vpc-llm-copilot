To specify the source resource, choose the resource type from Source type, and then choose the specific resource from Source. (Optional) You can filter the scope of the result based on the packet header leaving the source resource. For example, use the source and destination IP addresses and ports of interest. By default, the analysis considers all combinations of IP addresses and ports.To specify the destination resource, choose the resource type from Destination type, and then choose the specific resource from Destination. (Optional) You can filter the scope of the result based on the packet header arriving at the destination resource. For example, use the source and destination IP addresses and ports of interest. By default, the analysis considers all combinations of IP addresses and ports.For Protocol, choose TCP or UDP.(Optional) To add a tag, choose Add new tag and then enter the tag key and tag value.

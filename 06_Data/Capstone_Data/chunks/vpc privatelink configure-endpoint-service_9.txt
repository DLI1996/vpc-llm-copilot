To enable your endpoint service to accept IPv6 requests, its Network Load Balancers must use the dualstack IP address type. The targets do not need to support IPv6 traffic. For more information, see IP address type in the User Guide for Network Load Balancers.To modify the supported IP address types using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoint services.Select the VPC endpoint service.Choose Actions, Modify supported IP address types.

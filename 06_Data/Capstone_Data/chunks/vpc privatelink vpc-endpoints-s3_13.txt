You can change the route tables that are associated with the gateway endpoint. When you associate a route table, we automatically add a route that points traffic destined for the service to the endpoint network interface. When you disassociate a route table, we automatically remove the endpoint route from the route table.To associate route tables using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoints.Select the gateway endpoint.Choose Actions, Manage route tables.Select or deselect route tables as needed.Choose Modify route tables.To associate route tables using the command linemodify-vpc-endpoint (AWS CLI)Edit-EC2VpcEndpoint (Tools for Windows PowerShell)Edit the VPC endpoint policy

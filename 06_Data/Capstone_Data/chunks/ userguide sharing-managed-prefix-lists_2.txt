If you are part of an organization in AWS Organizations, and sharing within your organization is enabled, consumers in your organization are automatically granted access to the shared prefix list. Otherwise, consumers receive an invitation to join the resource share and are granted access to the shared prefix list after accepting the invitation.You can create a resource share and share a prefix list that you own using the AWS RAM console, or the AWS CLI.To create a resource share and share a prefix list using the AWS RAM consoleFollow the steps in Create a resource share in the AWS RAM User Guide. For Select resource type, choose Prefix Lists, and then select the check box for your prefix list.To add a prefix list to an existing resource share using the AWS RAM console

A member is a network interface associated with a supported EC2 instance that receives multicast traffic. For information about supported instances, see Multicast Consideration in Amazon VPC Transit Gateways.After you add the members, use SearchTransitGatewayMulticastGroups to verify that the members were added to the transit gateway multicast group.Request ParametersThe following parameters are for this specific action.

Checks whether you have the required permissions for the action, without actually making the request, and provides an error response. If you have the required permissions, the error response is DryRunOperation. Otherwise, it is UnauthorizedOperation. Type: Boolean Required: NoResponse ElementsThe following elements are returned by the service.requestIdThe ID of the request. Type: StringvpcInformation about the VPC.

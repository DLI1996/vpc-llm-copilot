association.ip-owner-id - The owner of the Elastic IP address (IPv4) associated with the network interface. association.public-ip - The address of the Elastic IP address (IPv4) bound to the network interface. association.public-dns-name - The public DNS name for the network interface (IPv4). attachment.attach-time - The time that the network interface was attached to an instance. attachment.attachment-id - The ID of the interface attachment. attachment.delete-on-termination - Indicates whether the attachment is deleted when an instance is terminated.

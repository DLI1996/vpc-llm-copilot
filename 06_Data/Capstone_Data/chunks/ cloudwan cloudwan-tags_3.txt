Remove or add tags as needed. To remove a tag, choose Remove tag, choose Remove tag for the tag you want to delete, and then choose Edit tags. To add a tag, choose Add tags, and then choose Add tag to add a new key-value pair. Or edit the Value of any existing tag. Choose Edit tags when finished. If the change that you made to the tags requires a tag acceptance from the core network owner, you will see the new proposed tags in the Proposed Tags tab.To remove a tagAccess the Network Manager console at https://console.aws.amazon.combetworkmanager/home/.Under Connectivity, choose Global Networks.

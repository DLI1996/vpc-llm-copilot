RFC 1918 ranges are the private IPv4 address ranges specified in RFC 1918.IPv6 VPC CIDR blocksYou can associate a single IPv6 CIDR block when you create a new VPC or you can associate up to five IPv6 CIDR blocks by modifying an existing VPC. The CIDR block is a fixed prefix length of /56. You can request an IPv6 CIDR block from Amazon's pool of IPv6 addresses. For more information, see Add an IPv6 CIDR block to your VPC.If you've associated an IPv6 CIDR block with your VPC, you can associate an IPv6 CIDR block with an existing subnet in your VPC or when you create a new subnet. For more information, see Subnet sizing for IPv6.

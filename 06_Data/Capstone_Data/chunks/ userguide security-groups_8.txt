Choose Actions, Manage tags. The Manage tags page displays any tags that are assigned to the security group.To add a tag, choose Add new tag and enter the tag key and tag value. To delete a tag, choose Remove next to the tag to delete.Choose Save changes.To tag a security group using the AWS CLIUse the create-tags command.Delete a security groupYou can delete a security group only if it is not associated with any resources. You can't delete a default security group.If you're using the console, you can delete more than one security group at a time. If you're using the command line or the API, you can delete only one security group at a time.To delete a security group using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Security groups.Select the security group and choose Actions, Delete security groups.

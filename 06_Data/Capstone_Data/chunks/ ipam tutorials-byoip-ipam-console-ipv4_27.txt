Run the following command to view the CIDR provisioned in the public IPv4 pool. aws ec2 describe-public-ipv4-pools --region us-east-2 --max-results 10 --profile member-account In the output, you'll see the provisioned CIDR. By default the CIDR is not advertised, which means it's not publicly accessible over the internet.

Configure the subnets For Number of Availability Zones, choose 2, so that you can launch instances in multiple Availability Zones to improve resiliency. For Number of public subnets, choose 2. For Number of private subnets, choose 2. You can keep the default CIDR block for the public subnet, or alternatively you can expand Customize subnet CIDR blocks and enter a CIDR block. For more information, see Subnet CIDR blocks.For NAT gateways, choose 1 per AZ to improve resiliency.If your application communicates by using IPv6 addresses, for Egress only internet gateway, choose Yes.

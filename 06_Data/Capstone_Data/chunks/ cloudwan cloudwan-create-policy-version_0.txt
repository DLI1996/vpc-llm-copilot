AWSDocumentationAWS Network ManagerAWS Cloud WAN User GuideCreate a policy version using the AWS Cloud WAN consoleCreate a policy version using the JSON editorCreate a core network policy versionYou can create a core network policy version at any time from the console, using visual editor mode or JSON mode. When you create a policy version, you can configure settings that determine how your network works. When you create a new policy version, a change set of the proposed core network changes is added. You can then review the changes and deploy the new core network and core network policy when you're ready.

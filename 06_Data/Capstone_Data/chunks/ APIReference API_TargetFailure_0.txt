Type: String Required: NoidThe ID of the target. If the target group type is INSTANCE, this is an instance ID. If the target group type is IP, this is an IP address. If the target group type is LAMBDA, this is the ARN of a Lambda function. If the target group type is ALB, this is the ARN of an Application Load Balancer. Type: String Required: NoportThe port on which the target is listening. This parameter doesn't apply if the target is a Lambda function. Type: Integer Valid Range: Minimum value of 1.

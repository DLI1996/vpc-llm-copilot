Access the Network Manager console at https://console.aws.amazon.combetworkmanager/home/.Under Connectivity choose Cloud WAN.On the Global networks page, choose the global network ID that for the core network you want to create a policy version for, and then choose Core network.In the navigation pane, choose Policy versions.Choose Create policy version.In Choose policy view mode, choose JSON.In the JSON editor, create your new policy. You can create a new policy version using a blank form, or copy and modify the contents of a policy version that you've downloaded. For the required and optional parameters in your JSON policy, see Core network policy parameters. For the steps to download a previous policy version, see Download a policy

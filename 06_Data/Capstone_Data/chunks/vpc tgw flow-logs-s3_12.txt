After you have created and configured your Amazon S3 bucket, you can create flow logs for transit gateways.To create a transit gateway flow log that publishes to Amazon S3 using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Transit gateways or Transit gateway attachments.Select the checkboxes for one or more transit gateways or transit gateway attachments.Choose Actions, Create flow log.Configure the flow log settings. For more information, see To configure flow log settings.To configure flow log settings using the consoleFor Destination, choose Send to an S3 bucket.

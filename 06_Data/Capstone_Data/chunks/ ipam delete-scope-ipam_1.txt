To delete a scope Open the IPAM console at https://console.aws.amazon.com/ipam/. In the navigation pane, choose Scopes. In the content pane, choose the scope that you want to delete. Choose Actions > Delete scope. Enter delete and then choose Delete.The commands in this section link to the AWS CLI Reference documentation. The documentation provides detailed descriptions of the options that you can use when you run the commands. Use the following AWS CLI commands to delete a scope: View scopes: describe-ipam-scopesDelete a scope: delete-ipam-scopeView updated scopes: describe-ipam-scopesTo create a new scope, see Create additional scopes. To delete the IPAM, see Delete an IPAM.

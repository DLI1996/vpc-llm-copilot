[Public NAT gateways only] The allocation ID of an Elastic IP address to associate with the NAT gateway. You cannot specify an Elastic IP address with a private NAT gateway. If the Elastic IP address is associated with another resource, you must first disassociate it. Type: String Required: NoClientTokenUnique, case-sensitive identifier that you provide to ensure the idempotency of the request. For more information, see How to ensure idempotency. Constraint: Maximum 64 ASCII characters. Type: String Required: NoConnectivityTypeIndicates whether the NAT gateway supports public or private connectivity. The default is public connectivity.

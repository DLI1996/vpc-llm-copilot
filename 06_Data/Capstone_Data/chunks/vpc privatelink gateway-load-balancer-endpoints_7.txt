Select the route table for the internet gateway and do the following: Choose Actions, Edit routes. If you support IPv4, choose Add route. For Destination, enter the IPv4 CIDR block of the subnet for the application servers. For Target, select the VPC endpoint. If you support IPv6, choose Add route. For Destination, enter the IPv6 CIDR block of the subnet for the application servers. For Target, select the VPC endpoint. Choose Save changes.

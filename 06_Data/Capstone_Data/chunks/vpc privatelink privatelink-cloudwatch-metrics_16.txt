To filter these metrics, use the following dimensions.Az Filters the metric data by Availability Zone. Load Balancer Arn Filters the metric data by load balancer. Service Id Filters the metric data by endpoint service. VPC Endpoint Id Filters the metric data by VPC endpoint.View the CloudWatch metricsYou can view these CloudWatch metrics using the Amazon VPC console, the CloudWatch console, or the AWS CLI as follows.To view metrics using the Amazon VPC consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoints. Select your endpoint and then choose the Monitoring tab.

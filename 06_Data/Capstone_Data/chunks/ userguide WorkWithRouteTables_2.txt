You can determine how many and which subnets or gateways are explicitly associated with a route table.The main route table can have explicit and implicit subnet associations. Custom route tables have only explicit associations.Subnets that aren't explicitly associated with any route table have an implicit association with the main route table. You can explicitly associate a subnet with the main route table. For an example of why you might do that, see Replace the main route table.To determine which subnets are explicitly associated using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Route tables.Check the Explicit subnet association column to determine the explicitly associated subnets and the Main column to determine whether this is the main route table.

edge-locations — An array of AWS Region locations where you're creating Core Network Edges. The array is composed of the following parameters: location — An AWS Region code, such as us-east-1. asn — (Optional) The ASN of the Core Network Edge in an AWS Region. By default, the ASN will be a single integer automatically assigned from asn-ranges. inside-cidr-blocks — (Optional) The local CIDR blocks for this Core Network Edge for AWS Transit Gateway Connect attachments.

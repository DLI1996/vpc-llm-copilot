Your account has a default quota of 20 gateway endpoints per Region, which is adjustable. There is also a limit of 255 gateway endpoints per VPC.Create a gateway endpointUse the following procedure to create a gateway endpoint that connects to DynamoDB.To create a gateway endpoint using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoints.Choose Create endpoint.For Service category, choose AWS services.For Services, add the filter Type: Gateway and select com.amazonaws.region.dynamodb.For VPC, select the VPC in which to create the endpoint.For Route tables, select the route tables to be used by the endpoint. We automatically add a route that points traffic destined for the service to the endpoint network interface.

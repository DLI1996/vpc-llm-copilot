Source or destination: The source (inbound rules) or destination (outbound rules) for the traffic to allow. Specify one of the following: A single IPv4 address. You must use the /32 prefix length. For example, 203.0.113.1/32. A single IPv6 address. You must use the /128 prefix length. For example, 2001:db8:1234:1a00::123/128. A range of IPv4 addresses, in CIDR block notation. For example, 203.0.113.0/24. A range of IPv6 addresses, in CIDR block notation. For example, 2001:db8:1234:1a00::/64. The ID of a prefix list. For example, pl-1234abc1234abc123. For more information, see Group CIDR blocks using managed prefix lists. The ID of a security group. For example, sg-1234567890abcdef0. For more information, see Security group referencing.

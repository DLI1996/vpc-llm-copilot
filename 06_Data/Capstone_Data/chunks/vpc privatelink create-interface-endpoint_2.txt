Create a VPC endpointUse the following procedure to create an interface VPC endpoint that connects to an AWS service.To create an interface endpoint for an AWS serviceOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Endpoints.Choose Create endpoint.For Service category, choose AWS services.For Service name, select the service. For more information, see AWS services that integrate with AWS PrivateLink.For VPC, select the VPC from which you'll access the AWS service.

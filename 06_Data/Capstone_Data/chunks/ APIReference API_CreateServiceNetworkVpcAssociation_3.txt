The IDs of the security groups. Security groups aren't added by default. You can add a security group to apply network level controls to control which resources in a VPC are allowed to access the service network and its services. For more information, see Control traffic to resources using security groups in the Amazon VPC User Guide. Type: Array of strings Array Members: Minimum number of 0 items. Maximum number of 5 items.

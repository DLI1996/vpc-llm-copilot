key — A string that matches against any of the keys defined on the attachment. It must be an exact match of the key. operator — The operation to perform against the key value. Must be one of equals | not-equals | contains | begins-with. value — The value of the key to be evaluated for the operator. In this example, “type” : “tag-value”, “key” : “project”, “operator” : “begins-with”, “value” : “sta”

Select the checkbox for the prefix list.In the lower pane, choose Associations to view the resources that are referencing the prefix list.To view prefix list associations using the AWS CLIUse the get-managed-prefix-list-associations command.Modify a prefix listYou can modify the name of your prefix list, and you can add or remove entries. To modify the maximum number of entries, see Resize a prefix list.Updating the entries of a prefix list creates a new version of the prefix list. Updating the name or maximum number of entries for a prefix list does not create a new version of the prefix list.ConsiderationsYou cannot modify an AWS-managed prefix list.

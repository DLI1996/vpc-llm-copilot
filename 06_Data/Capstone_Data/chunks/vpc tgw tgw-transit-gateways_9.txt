The details for the transit gateway are displayed below the list of gateways on the page.To view your transit gateways using the AWS CLIUse the describe-transit-gateways command.Add or edit tags for a transit gatewayAdd tags to your resources to help organize and identify them, such as by purpose, owner, or environment. You can add multiple tags to each transit gateway. Tag keys must be unique for each transit gateway. If you add a tag with a key that is already associated with the transit gateway, it updates the value of that tag.

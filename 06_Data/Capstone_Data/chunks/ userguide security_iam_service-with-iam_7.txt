All Amazon EC2 actions support the aws:RequestedRegion and ec2:Region condition keys. For more information, see Example: Restrict access to a specific Region.Amazon VPC defines its own set of condition keys and also supports using some global condition keys. To see a list of Amazon VPC condition keys, see Condition keys for Amazon EC2 in the Service Authorization Reference. To learn with which actions and resources you can use a condition key, see Actions defined by Amazon EC2.Amazon VPC resource-based policiesResource-based policies are JSON policy documents that specify what actions a specified principal can perform on the Amazon VPC resource and under what conditions.

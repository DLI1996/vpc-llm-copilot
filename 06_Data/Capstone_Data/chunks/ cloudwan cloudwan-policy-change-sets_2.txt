Choose Create policy. This creates a new version of the policy. The policy version is incremented by one from the last version. The Change set state of the new version is set to Pending generation on the Policy versions page, and the alias is set to LATEST, indicating that this is the most recent version of the policy. When a policy version has been generated, the Change set state changes to Ready to execute. You can then implement the new policy version as your LIVE policy.

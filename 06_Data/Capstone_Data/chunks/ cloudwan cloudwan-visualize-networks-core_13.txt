On the Topology graph page, you can view a topology diagram of your core network that includes core network and transit gateway networks. It includes information about AWS Regions, core network edges, segments, VPCs, VPNs, and Connect attachments. Icons represent specific resource type and lines represent connections between resources. The line colors represent the state of the connection between AWS and the on-premises resources.

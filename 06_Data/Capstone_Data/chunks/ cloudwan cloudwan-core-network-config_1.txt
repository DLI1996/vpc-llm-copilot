On the Global networks page, choose the global network ID that for the core network you want to create a policy version for, and then choose Core network.In the navigation pane, choose Policy versions.Choose Create policy version.In Choose policy view mode, choose Visual editor.The Network configuration displays general settings for the policy.In General settings, choose Edit. The Version can't be changed for a policy version. Choose VPN ECMP support if the core network should forward traffic over multiple-cost routes using VPN. Choose Edit general settings.

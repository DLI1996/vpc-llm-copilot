If you've shared the prefix list using AWS RAM, any references in consumer-owned resources must first be removed.LimitationYou cannot delete an AWS-managed prefix list.To delete a prefix list using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Managed Prefix Lists.Select the prefix list, and choose Actions, Delete prefix list.In the confirmation dialog box, enter delete, and choose Delete.To delete a prefix list using the AWS CLIUse the delete-managed-prefix-list command.Javascript is disabled or is unavailable in your browser.

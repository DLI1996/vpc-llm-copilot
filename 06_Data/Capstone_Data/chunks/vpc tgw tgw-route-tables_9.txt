On the navigation pane, choose Transit Gateway Route Tables.Choose the route table that includes the routes to export.Choose Actions, Export routes.On the Export routes page, for S3 bucket name, type the name of the S3 bucket.To filter the routes exported, specify filter parameters in the Filters section of the page.Choose Export routes.To access the exported routes, open the Amazon S3 console at https://console.aws.amazon.com/s3/, and navigate to the bucket that you specified. The file name includes the AWS account ID, AWS Region, route table ID, and a timestamp. Select the file and choose Download.

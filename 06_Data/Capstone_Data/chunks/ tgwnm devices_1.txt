On the Global networks page, choose the global network ID.In the navigation pane, choose Devices. Choose Create device.For Name and Description, enter a name and description for the device.For Model, enter the device model number.For Serial number, enter the serial number for the device.For Type, enter the device type.For Vendor, enter the name of the vendor, for example, Cisco.For Location type, specify whether the device is located in a remote location (on-premises network, data center, or other cloud provider) or in AWS. If you choose AWS Cloud, specify the location of the device within AWS. For Zone, specify the name of an Availability Zone, Local Zone, Wavelength Zone, or an Outpost. For Subnet, specify the Amazon Resource Name (ARN) of a subnet (for example, arn:aws:ec2:us-east-1:111111111111:subnet/subnet-abcd1234).

When you delete a rule from a security group, the change is automatically applied to any instances that are associated with the security group.To delete a security group rule using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.In the navigation pane, choose Security groups.Select the security group.Choose Actions, and then choose Edit inbound rules to remove an inbound rule or Edit outbound rules to remove an outbound rule.Choose the Delete button next to the rule to delete.Choose Save rules. Alternatively, choose Preview changes, review your changes, and choose Confirm.To delete a security group rule using the AWS CLIUse the revoke-security-group-ingress and revoke-security-group-egress commands.Example rulesWeb servers

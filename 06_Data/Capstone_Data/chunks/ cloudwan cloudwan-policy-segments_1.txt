On the Global networks page, choose the global network ID that for the core network you want to create a policy version for, and then choose Core network.In the navigation pane, choose Policy versions.Choose Create policy version.Choose Segments.In the Segments section, Choose Create.Enter the Segment name and Segment description to identify the segment.From the Edge locations dropdown list, choose one or more segments to create.Choose Require acceptance if you require approval for attachments to be mapped to this segment.Choose Isolated attachments if you need this segment isolated. Attachments in isolated segments can't communicate with other segments, and attachments in other segments can't communicate with the isolated segment.

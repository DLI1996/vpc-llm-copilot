If you do not use the middlebox routing wizard, you must manually configure, and then assign the route tables to the subnets and internet gateway.Internet gateway route tableThe route table for the internet gateway has the following routes.Consumer VPC CIDR Local Local route Application subnet CIDR endpoint-id Routes traffic destined for the application subnet to the Gateway Load Balancer endpoint.There is an edge association with the gateway.When you use the middlebox routing wizard, it associates the following tags with the route table:The key is "Origin" and the value is "Middlebox wizard"The key is "date_created" and the value is the creation time (for example, "2021-02-18T22:25:49.137Z")Application subnet route table

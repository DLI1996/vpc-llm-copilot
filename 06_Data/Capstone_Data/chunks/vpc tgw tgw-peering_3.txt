Before you begin, ensure that you have the ID of the transit gateway that you want to attach. If the transit gateway is in another AWS account, ensure that you have the AWS account ID of the owner of the transit gateway.After you create the peering attachment, the owner of the accepter transit gateway must accept the attachment request.To create a peering attachment using the consoleOpen the Amazon VPC console at https://console.aws.amazon.com/vpc/.On the navigation pane, choose Transit Gateway Attachments.Choose Create transit gateway attachment.For Transit gateway ID, choose the transit gateway for the attachment. You can choose a transit gateway that you own or a transit gateway that was shared with you.For Attachment type, choose Peering Connection.Optionally enter a name tag for the attachment.

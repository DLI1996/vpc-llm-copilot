AWSDocumentationAmazon VPCUser GuideControl traffic to your AWS resources using security groupsA security group controls the traffic that is allowed to reach and leave the resources that it is associated with. For example, after you associate a security group with an EC2 instance, it controls the inbound and outbound traffic for the instance. You can associate a security group only with resources in the VPC for which it is created.When you create a VPC, it comes with a default security group. You can create additional security groups for each VPC.There is no additional charge for using security groups.

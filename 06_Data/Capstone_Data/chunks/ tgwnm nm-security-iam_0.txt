AWSDocumentationAWS Network ManagerAWS Global Networks for Transit Gateways User GuideHow Network Manager works with IAMExample policiesIdentity and access management for AWS Global Networks for Transit GatewaysAWS Identity and Access Management (IAM) is an AWS service that helps an administrator securely control access to AWS resources. IAM administrators control who can be authenticated (signed in) and authorized (have permissions) to use AWS Global Networks for Transit Gateways resources. IAM is an AWS service that you can use with no additional charge. You can use features of IAM to allow other users, services, and applications to use your AWS resources fully or in a limited way, without sharing your security credentials.By default, IAM users don't have permission to create, view, or modify AWS resources.

They illustrate that each pool within the top-level pool is provisioned with a portion of the top-level CIDR.To create a development pool within a Regional poolOpen the IPAM console at https://console.aws.amazon.com/ipam/. In the navigation pane, choose Pools. Choose Create pool. Under IPAM scope, choose the same scope that you used when you created the top-level and Regional pools. For more information about scopes, see How IPAM works. (Optional) Add a Name tag for the pool and a description for the pool.

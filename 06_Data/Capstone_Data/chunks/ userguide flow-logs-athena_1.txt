You can streamline and automate the integration of your VPC flow logs with Athena by generating a CloudFormation template that creates the required AWS resources and predefined queries that you can run to obtain insights about the traffic flowing through your VPC.You can create your own queries using Athena. For more information, see Query flow logs using Amazon Athena in the Amazon Athena User Guide.PricingYou incur standard Amazon Athena charges for running queries.

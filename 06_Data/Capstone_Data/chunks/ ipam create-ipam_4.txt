To create an IPAM Open the IPAM console at https://console.aws.amazon.com/ipam/. In the AWS Management Console, choose the AWS Region in which you want to create the IPAM. Create the IPAM in your main Region of operations. On the service home page, choose Create IPAM. Select Allow Amazon VPC IP Address Manager to replicate data from source account(s) into the IPAM delegate account. If you do not select this option, you cannot create an IPAM.

Configure the alarm as follows, and then choose Next: For Statistic, choose Maximum. For Period, choose 5 minutes. For Whenever, choose Greater and enter 0 for the threshold. For Additional configuration, Datapoints to alarm, enter 3.For Notification, select an existing SNS topic or choose Create new topic to create a new one. Choose Next.Enter a name and description for the alarm and choose Next.When you are done configuring the alarm, choose Create alarm.For more information, see Using Amazon CloudWatch alarms in the Amazon CloudWatch User Guide.

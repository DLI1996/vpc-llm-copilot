NTP servers: The NTP servers that provide the time to the instances.NetBIOS name servers: For EC2 instances running Windows, the NetBIOS computer name is a friendly name assigned to the instance to identify it on the network. A NetBIOS name server maintains a list of mappings between NetBIOS computer names and network addresses for networks that use NetBIOS as their naming service.NetBIOS node type: For EC2 instances running Windows, the method that the instances use to resolve NetBIOS names to IP addresses.

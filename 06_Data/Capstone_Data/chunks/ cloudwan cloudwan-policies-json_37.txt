action — The action to take when a condition is true. association-method — Defines how a segment is mapped. Values can be constant or tag. constant statically defines the segment to associate the attachment to. tag uses the value of a tag to dynamically try to map to a segment. segment — The name of the segment to share as defined in the segments section. This is used only when the association-method is constant. tag-value-of-key — Maps the attachment to the value

The default BGP keep-alive timeout is 10 seconds and the default hold timer is 30 seconds.IPv6 BGP peering is not supported; only IPv4-based BGP peering is supported. IPv6 prefixes are exchanged over IPv4 BGP peering using MP-BGP.Bidirectional Forwarding Detection (BFD) is not supported.BGP graceful restart is not supported.When you create a transit gateway peer, if you do not specify a peer ASN number, we pick the transit gateway ASN number. This means that your appliance and transit gateway will be in the same autonomous system doing iBGP.

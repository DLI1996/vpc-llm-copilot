From your local computer, configure SSH agent forwarding, so that you can use the NAT instance as a bastion server. Linux and macOS ssh-add key.pem Windows Download and install Pageant, if it is not already installed. Convert your private key to .ppk format using PuTTYgen. Start Pageant, right-click the Pageant icon on the taskbar (it might be hidden), and choose Add Key. Select the .ppk file that you created, enter the passphrase if required, and choose Open.

In the Edge locations section, do the following: Choose Create. From the Location dropdown list, choose the Region where you want the Core Network Edge router to be created. You can choose only one Region. For ASN, enter the ASN number for the Region. For Inside CIDR block, enter the CIDR block that you want to use for BGP peering on Connect peers.

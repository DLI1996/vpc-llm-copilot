To manually allocate a CIDROpen the IPAM console at https://console.aws.amazon.com/ipam/. In the navigation pane, choose Pools. By default, the default private scope is selected. If you donâ€™t want to use the default private scope, from the dropdown menu at the top of the content pane, choose the scope you want to use. For more information about scopes, see How IPAM works. In the content pane, choose a pool. Choose Actions > Create custom allocation. Choose whether to add a specific CIDR to allocate (for example, 10.0.0.0/24 for IPv4 or 2001:db8::/52 for IPv6) or add a CIDR by size by choosing the netmask length only (for example, /24 for IPv4 or /52 for IPv6). Choose Allocate.

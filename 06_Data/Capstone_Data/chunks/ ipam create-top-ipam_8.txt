To provision a CIDR, choose Add new CIDR. Enter an IPv4 CIDR to provision for the pool. If you want to bring your own IPv4 or IPv6 IP address range to AWS there are prerequisites. For more information, see Tutorial: Bring your IP addresses to IPAM. Choose optional allocation rules for this pool: Automatically import discovered resources: This option is not available if the Locale is set to None.

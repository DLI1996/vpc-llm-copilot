If you release an Elastic IP address but change your mind, you might be able to recover it. You cannot recover the Elastic IP address if it has been allocated to another AWS account, or if recovering it results in you exceeding your Elastic IP address quota.You can recover an Elastic IP address by using the Amazon EC2 API or a command line tool.To recover an Elastic IP address using the AWS CLIUse the allocate-address command and specify the IP address using the --address parameter.API and command overviewYou can perform the tasks described in this section using the command line or an API.

To create a Regional pool within a top-level poolOpen the IPAM console at https://console.aws.amazon.com/ipam/. In the navigation pane, choose Pools. Choose Create pool. Under IPAM scope, choose the same scope that you used when you created the top-level pool. For more information about scopes, see How IPAM works. (Optional) Add a Name tag for the pool and a description for the pool. Under Source type, choose IPAM pool. Then choose the top-level pool that you created in the previous section.

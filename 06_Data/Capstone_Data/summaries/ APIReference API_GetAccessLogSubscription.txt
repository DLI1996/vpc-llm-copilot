This documentation is about the AWS API for retrieving information about a specified access log subscription in Amazon VPC Lattice. The request syntax includes URI request parameters and does not have a request body. The response syntax includes various response elements such as the ARN, creation date, destination ARN, ID, last update date, resource ARN, and resource ID. The API may return errors such as AccessDeniedException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources and SDKs for different programming languages.

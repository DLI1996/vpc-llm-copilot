This document provides information about the common parameters used for signing Signature Version 4 requests with a query string in AWS Global Networks for Transit Gateways. The parameters include the action to be performed, the API version, the hash algorithm used for the request signature, the credential scope value, the date used to create the signature, the temporary security token obtained from AWS STS, the signature, and the signed headers. These parameters are required under certain conditions and are used for authentication purposes. The document also mentions that JavaScript must be enabled to use the AWS Documentation.

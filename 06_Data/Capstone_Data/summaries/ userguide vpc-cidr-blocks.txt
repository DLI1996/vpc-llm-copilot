This documentation provides information about managing IPv4 and IPv6 CIDR blocks for an Amazon Virtual Private Cloud (VPC) in AWS. It explains how to create a VPC with an IPv4 CIDR block and associate additional CIDR blocks. It also discusses the restrictions and recommendations for choosing CIDR blocks, as well as managing CIDR blocks for VPC peering connections and Direct Connect gateways. Additionally, it covers the process of associating and disassociating IPv6 CIDR blocks with a VPC.

This documentation is about the AWS Global Networks for Transit Gateways API, specifically the SiteToSiteVpnAttachment API. It provides information on how to create a site-to-site VPN attachment and the details required for it. The documentation also includes the ARN (Amazon Resource Name) of the site-to-site VPN attachment and its specifications. Additionally, it provides links to language-specific AWS SDKs for further information on using the API.

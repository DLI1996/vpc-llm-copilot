This document provides instructions on how to use multicast with AWS Transit Gateway. It includes steps for creating an IGMP multicast domain and a static source multicast domain, associating VPC attachments and subnets with a multicast domain, viewing multicast domain associations, disassociating subnets from a multicast domain, adding tags to a multicast domain, and deleting a multicast domain. The document also provides information on managing multicast domains and includes links to additional resources for working with multicast.

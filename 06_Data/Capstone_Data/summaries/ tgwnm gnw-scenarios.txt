The AWS Global Networks for Transit Gateways User Guide provides documentation on how to manage transit gateways in AWS. It covers various scenarios such as managing multi-region and multi-account global networks, single devices with single or multiple VPN connections, multi-device and multi-link sites, SD-WAN connections to AWS, and connections between devices. The guide provides step-by-step instructions on creating global networks, registering transit gateways, creating sites, devices, and links, associating customer gateways, and managing Transit Gateway Connect peers.

This section provides a list of quotas related to IP Address Manager (IPAM) in Amazon VPC. It includes quotas for IPAM administrators per organization, IPAMs per region, scopes per IPAM, pools per scope, CIDRs per pool, pool depth, Amazon-provided IPv6 CIDR blocks per regional pool, Amazon-provided IPv6 CIDR block netmask length, resource discoveries per region, resource discovery associations per IPAM, and resource utilization metrics. Some quotas can be adjusted upon request, while others cannot. For more information on requesting quota increases, refer to the Service Quotas User Guide.

This is a summary of the AWS documentation for the Amazon VPC Lattice API reference for the BatchUpdateRule operation. This operation allows you to update listener rules in a batch, including changing the priority of the rules. The operation requires the vpc-lattice:UpdateRule permission. The request includes URI parameters for the listener identifier and service identifier, as well as a request body with an array of RuleUpdate objects. The response includes successful and unsuccessful updates, with corresponding objects. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException.

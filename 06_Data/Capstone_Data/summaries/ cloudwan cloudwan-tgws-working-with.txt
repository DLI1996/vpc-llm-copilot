The AWS Cloud WAN User Guide provides documentation on AWS Network Manager, which is a service that allows users to manage and monitor their network resources. The guide covers various topics such as the overview of transit gateways, topology tree, events, monitoring, on-premises associations, connect peer, and tags. It provides step-by-step instructions on how to access and view dashboard information, track events, monitor transit gateways, manage on-premises associations and connect peers, and view and edit tags associated with transit gateways.

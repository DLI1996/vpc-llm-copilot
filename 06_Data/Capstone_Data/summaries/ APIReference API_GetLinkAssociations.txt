This documentation is about the AWS Global Networks for Transit Gateways API, specifically the GetLinkAssociations operation. This operation retrieves the link associations for a device or a link. The request can be made using URI parameters such as DeviceId, globalNetworkId, LinkId, MaxResults, and NextToken. The response includes the link associations and a NextToken for pagination. The documentation also provides information about possible errors and references to other related APIs and SDKs.

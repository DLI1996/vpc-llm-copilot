This is a summary of the documentation for the AWS API action "DeleteInternetGateway". This action is used to delete a specified internet gateway. The request parameters include "DryRun" (a boolean value to check permissions) and "InternetGatewayId" (the ID of the internet gateway). The response elements include "requestId" (the ID of the request) and "return" (a boolean value indicating success or error). The documentation also provides examples and references to other AWS SDKs for different programming languages.

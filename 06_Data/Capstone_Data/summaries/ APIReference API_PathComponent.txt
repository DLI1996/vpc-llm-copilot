This is a summary of the documentation for AWS Global Networks for Transit Gateways. It provides information about the PathComponent, which describes a path component. The PathComponent includes the DestinationCidrBlock, which is the destination CIDR block in the route table, and the Resource, which is the resource associated with the path. It also includes the Sequence, which is the sequence number in the path. The documentation provides links to language-specific AWS SDKs for more information on using this API.

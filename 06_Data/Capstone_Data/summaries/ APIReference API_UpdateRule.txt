This documentation is about the AWS API operation "UpdateRule" which is used to update a rule for a listener. The request syntax includes parameters such as listenerIdentifier, ruleIdentifier, and serviceIdentifier. The request body includes information about the action, match, and priority of the rule. The response syntax includes information about the action, ARN, ID, isDefault, match, name, and priority of the rule. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException.

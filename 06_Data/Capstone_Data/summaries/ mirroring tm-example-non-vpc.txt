This documentation provides instructions on how to set up traffic mirroring in Amazon VPC. It outlines the steps to create a traffic mirror target, a traffic mirror filter, and a traffic mirror session. It also provides an example of how to mirror non-local VPC traffic. The example includes creating a traffic mirror target for a monitoring appliance, creating a traffic mirror filter with rules for outbound and inbound traffic, and creating a traffic mirror session with the appropriate source, target, and filter.

This documentation is about the AWS Global Networks for Transit Gateways API, specifically the CreateLink operation. It provides information on the request syntax, URI request parameters, request body, response syntax, response elements, and possible errors. The CreateLink operation is used to create a new link for a specified site, with parameters such as bandwidth, description, provider, site ID, tags, and type. The response includes information about the created link. The documentation also provides links to related API operations and SDKs for different programming languages.

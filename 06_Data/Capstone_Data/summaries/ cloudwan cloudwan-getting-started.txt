The AWS Cloud WAN User Guide provides instructions on how to set up and manage a global network using AWS Cloud WAN. The guide outlines the prerequisites for setting up AWS Cloud WAN and provides step-by-step instructions for creating a global network, core network, attachments, devices, and sites. It also explains how to register transit gateways and work with and modify the network once it is set up. The guide includes information on visualizing and monitoring the network and provides links to additional resources for further assistance.

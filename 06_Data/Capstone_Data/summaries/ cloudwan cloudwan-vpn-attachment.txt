This documentation provides instructions on how to add a Site-to-Site VPN attachment to your core network edge using AWS Network Manager. It explains that you can add the attachment either through the console or using the command line/API. It also mentions that the Site-to-Site VPN must be attached to a core network before configuring a customer gateway, and that the attachment must be created in the same AWS account that owns the core network. The documentation provides step-by-step instructions for adding the attachment using the console and also mentions that it can be done using the command line or API.

This is a summary of the documentation for the AWS API operation "UpdateTargetGroup". This operation updates the specified target group. The request includes parameters such as health check configuration and the target group identifier. The response includes the ARN, configuration, ID, name, status, and type of the target group. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources for using this API in different programming languages.

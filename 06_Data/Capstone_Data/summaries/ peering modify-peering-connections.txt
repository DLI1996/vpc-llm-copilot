This documentation explains how to enable DNS resolution for a VPC peering connection in AWS. It provides step-by-step instructions on how to modify the peering connection options to enable DNS resolution. It also mentions that both VPCs involved in the peering connection must be enabled for DNS hostnames and DNS resolution. The documentation includes instructions for enabling DNS resolution through the AWS console, command line, or API. It also mentions that for inter-Region peering connections, the requester VPC and accepter VPC peering options must be modified in their respective Regions.

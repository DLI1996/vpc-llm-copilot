The AWS Network Manager allows users to visualize and monitor their transit gateway networks. The documentation provides an overview of the features and functionalities of the Network Manager, including the ability to view details about transit gateways, monitor VPN and connect peer status, track network events, analyze routes, and monitor network usage metrics. Users can access these features through the Network Manager console and customize their monitoring settings. The documentation also provides instructions on how to access and navigate the different sections of the Network Manager, such as the Overview, Geography, Topology Tree, Events, Monitoring, and Route Analyzer pages. Additionally, it explains how to switch roles to view resource details in member accounts and provides information on using CloudWatch Events and CloudWatch Dashboards for monitoring purposes.

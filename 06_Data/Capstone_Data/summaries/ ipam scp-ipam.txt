This documentation provides instructions on how to enforce IP Address Manager (IPAM) when creating Virtual Private Clouds (VPCs) in AWS. It includes steps to create a service control policy (SCP) in AWS Organizations that requires the use of IPAM for VPC creation. It also provides instructions on how to enforce the use of a specific IPAM pool or enforce IPAM for all but a given list of Organizational Units (OUs). The document includes JSON code examples and guidance on attaching the policy to the appropriate organizational units.

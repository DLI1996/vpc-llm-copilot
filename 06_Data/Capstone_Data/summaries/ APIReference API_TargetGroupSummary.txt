This is a summary of the documentation for the TargetGroupSummary API in Amazon VPC Lattice. The TargetGroupSummary provides summary information about a target group. It includes details such as the ARN, creation date, ID, IP address type, Lambda event structure version, last update date, name, port, protocol, service ARNs, status, type, and VPC identifier. The documentation also provides information on the valid values and constraints for each parameter. Additionally, it includes links to the AWS SDKs for different programming languages and mentions that JavaScript must be enabled to use the AWS Documentation.

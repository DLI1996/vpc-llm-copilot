This documentation is about the AWS Global Networks for Transit Gateways API, specifically the DeleteDevice operation. This operation is used to delete an existing device by providing the device ID and global network ID as URI parameters. The request does not have a request body. The response includes information about the deleted device in JSON format. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to related APIs and SDKs for different programming languages.

This document provides instructions on how to register a transit gateway in AWS Cloud WAN. It states that a transit gateway must first be created on the Amazon Virtual Private Cloud console. The steps to create a transit gateway can be found in the Amazon VPC Transit Gateways Guide. To register a transit gateway in AWS Cloud WAN, the user needs to access the Network Manager console, choose Global Networks, select the global network ID, choose Transit gateways, select the transit gateway to register, and then choose Register Transit Gateway. The document also mentions an optional step to create a site.

This documentation is for the AWS API reference for the DeleteTransitGatewayRouteTableAnnouncement action. It provides information about the request parameters, response elements, errors, and related resources. The DeleteTransitGatewayRouteTableAnnouncement action is used to advertise to the transit gateway that a transit gateway route table is deleted. The documentation also includes links to language-specific AWS SDKs and information about enabling JavaScript to access the AWS documentation.

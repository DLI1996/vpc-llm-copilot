This documentation is about the AWS Global Networks for Transit Gateways API, specifically the GetCustomerGatewayAssociations operation. This operation retrieves the association information for customer gateways that are associated with devices and links in your global network. The request can include parameters such as customer gateway ARNs, global network ID, maximum number of results, and a token for pagination. The response includes customer gateway associations and a token for the next page of results. The documentation also provides information about possible errors and links to other related APIs and SDKs.

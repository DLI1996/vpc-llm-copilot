This documentation is about the AWS Global Networks for Transit Gateways API, specifically the DeleteLink operation. This operation is used to delete an existing link in the global network. The request requires the global network ID and link ID as URI parameters. The response includes information about the deleted link. Possible errors include access denied, conflict, internal server error, resource not found, throttling, and validation failure. This API can be used with various AWS SDKs and command line interfaces.

The AWS ListServiceNetworkVpcAssociations API allows you to list the service network and VPC associations. You can filter the list by either the VPC or service network identifier. The request parameters include maxResults, nextToken, serviceNetworkIdentifier, and vpcIdentifier. The response includes information about the associations and a pagination token for the next page of results. Possible errors include AccessDeniedException, InternalServerException, ThrottlingException, and ValidationException. This API can be used with various AWS SDKs.

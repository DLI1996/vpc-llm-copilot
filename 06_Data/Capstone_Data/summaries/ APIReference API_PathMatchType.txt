This is a summary of the documentation for the Amazon VPC Lattice API Reference. It describes the PathMatchType data type, which is used to specify the type of path match in a rule. The PathMatchType can be either "exact" or "prefix". The documentation provides information on the length constraints and pattern for each type. It also includes links to the AWS SDKs for different programming languages. The documentation requires JavaScript to be enabled in order to be used. Users are encouraged to provide feedback on the documentation to help improve it.

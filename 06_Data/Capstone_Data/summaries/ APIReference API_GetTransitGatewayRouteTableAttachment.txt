This documentation is about the AWS Global Networks for Transit Gateways API, specifically the GetTransitGatewayRouteTableAttachment operation. It provides information about a transit gateway route table attachment. The request requires the attachmentId parameter and does not have a request body. The response includes information about the transit gateway route table attachment. Possible errors include AccessDeniedException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources for using the API in different programming languages.

This is a summary of the documentation for the AWS API action "AttachInternetGateway". This action allows you to attach an internet gateway or a virtual private gateway to a VPC, enabling connectivity between the internet and the VPC. The action requires the parameters "InternetGatewayId" and "VpcId". The response includes the "requestId" and "return" elements. The documentation also provides examples and references to other related APIs and SDKs.

This is a summary of the documentation for the Amazon VPC Lattice API reference. It provides information about the TargetSummary, which includes the ID, port, reason code, and status of a target. The status can be DRAINING, UNAVAILABLE, HEALTHY, UNHEALTHY, INITIAL, or UNUSED. The documentation also includes links to language-specific AWS SDKs and information about enabling JavaScript to access the Amazon Web Services Documentation.

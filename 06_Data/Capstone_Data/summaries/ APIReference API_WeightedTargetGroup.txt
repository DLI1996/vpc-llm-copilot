This is a summary of the documentation for the WeightedTargetGroup API in Amazon VPC Lattice. The WeightedTargetGroup API is used to describe the weight of a target group. The API requires the targetGroupIdentifier parameter, which is the ID or ARN of the target group. The weight parameter is optional and is used to determine how requests are distributed to the target group. The API supports various AWS SDKs for different programming languages. The documentation also provides information on common parameters and document conventions.

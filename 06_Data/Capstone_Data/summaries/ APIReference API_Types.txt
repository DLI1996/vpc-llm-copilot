The AWS Network Manager API documentation provides information on various data types used in the API. It includes details about each data type and their elements. The documentation also mentions that the order of elements in a data type structure is not guaranteed. The API supports several data types such as AccountStatus, Attachment, Bandwidth, BgpOptions, Connection, and many more. The documentation also mentions that JavaScript must be enabled to use the AWS documentation. Users are encouraged to provide feedback on the documentation to help improve it.

This is a brief summary of the AWS Global Networks for Transit Gateways API Reference documentation. It describes the ConnectAttachmentOptions, which includes information about the protocol used for the attachment connection. The valid values for the protocol are GRE and NO_ENCAP. The documentation also provides links to language-specific AWS SDKs for C++, Go, Java V2, and Ruby V3. It mentions that Javascript must be enabled to use the Amazon Web Services Documentation. The page also asks for feedback on whether the documentation was helpful or not and provides an opportunity to suggest improvements.

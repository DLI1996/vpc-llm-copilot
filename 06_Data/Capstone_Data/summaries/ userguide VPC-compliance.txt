This document provides information on compliance validation for Amazon Virtual Private Cloud (VPC) and the resources available to help with compliance when using AWS services. It includes links to AWS services in scope for specific compliance programs, third-party audit reports, security and compliance quick start guides, resources for HIPAA security and compliance, AWS Compliance Resources, evaluating resources with rules in AWS Config, AWS Security Hub, and AWS Audit Manager. It also mentions that JavaScript must be enabled to use the Amazon Web Services Documentation.

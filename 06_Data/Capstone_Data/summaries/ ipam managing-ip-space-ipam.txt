This documentation is about managing IP address space in Amazon VPC using an IP Address Manager (IPAM). It provides optional tasks for IPAM administrators, including enforcing IPAM use for VPC creation, sharing IPAM pools using AWS RAM, provisioning and deprovisioning CIDRs to/from a pool, editing and deleting a pool, working with resource discoveries, creating additional scopes, moving VPC CIDRs between scopes, changing the monitoring state of VPC CIDRs, releasing an allocation, and deleting an IPAM. The documentation also mentions that JavaScript must be enabled to use the Amazon Web Services Documentation.

The AWS API reference provides documentation for the DeleteServiceNetwork API, which is used to delete a service network in Amazon VPC Lattice. The request requires the serviceNetworkIdentifier parameter, which is the ARN or ID of the service network. The response is an HTTP 204 with an empty body. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The API can be used with various AWS SDKs.

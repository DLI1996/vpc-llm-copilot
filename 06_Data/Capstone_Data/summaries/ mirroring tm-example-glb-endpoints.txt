This documentation provides a step-by-step guide on how to set up traffic mirroring in AWS using Amazon VPC. It includes instructions on creating traffic mirror targets, filters, and sessions in different VPCs. It also provides an example of mirroring traffic to appliances behind a Gateway Load Balancer using Gateway Load Balancer endpoints. The documentation concludes with information on how to mirror non-local VPC traffic and additional resources for working with traffic mirroring.

This documentation is for the AWS API action "GetTransitGatewayAttachmentPropagations". It lists the route tables to which a specified resource attachment propagates routes. The action has several request parameters, including "DryRun" to check permissions, "Filter.N" for filtering results, "MaxResults" to limit the number of results, and "NextToken" for pagination. The response includes the "nextToken" for retrieving the next page of results, the "requestId", and information about the propagation route tables. Common errors and language-specific SDKs are also mentioned.

This documentation is about the AWS API for deleting a listener rule in Amazon VPC Lattice. The request syntax includes URI request parameters such as listenerIdentifier, ruleIdentifier, and serviceIdentifier. The response syntax is an HTTP 204 response with an empty body if the action is successful. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException. The documentation also provides links to additional resources and language-specific AWS SDKs.

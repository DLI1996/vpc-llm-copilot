The AWS API Reference for the DescribeTransitGatewayAttachments action provides information about attachments between resources and transit gateways. It describes the request parameters, response elements, and possible errors. The action allows filtering results by attachment ID, attachment state, resource ID, or resource owner. The maximum number of results that can be returned in a single call is specified by the MaxResults parameter. The response includes a nextToken value for retrieving the next page of results.

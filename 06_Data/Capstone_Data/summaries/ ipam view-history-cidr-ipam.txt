This documentation explains how to view the history of an IP address or CIDR in an IP Address Manager (IPAM) scope in Amazon VPC. The historical data can be used for network security and routing policy analysis. The IPAM retains IP address monitoring data for up to three years. The documentation provides step-by-step instructions on how to search for the status change of IP addresses or CIDRs for various types of resources such as VPCs, VPC subnets, Elastic IP addresses, EC2 instances, and EC2 network interfaces. It also provides information on how the history records are created and managed when resources are moved or transferred. The documentation includes details on the search results and the meanings of different columns in the search results.

This documentation is about the AWS API action "DisableAwsNetworkPerformanceMetricSubscription" which is used to disable infrastructure performance metric subscriptions. The action requires parameters such as the target region or availability zone, whether to perform a dry run, the metric and statistic used for the disabled subscription. The response includes information about the success of the action and the request ID. The documentation also provides information about common errors and references to language-specific AWS SDKs.

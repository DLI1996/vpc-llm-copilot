This is a summary of the AWS API reference for the DescribeSecurityGroupReferences action. This action allows you to retrieve information about the VPCs that are referencing the specified security groups. The request parameters include the option to perform a dry run and the IDs of the security groups. The response includes the request ID and information about the VPCs with the referencing security groups. The documentation also provides examples and references to other AWS SDKs for using this API.

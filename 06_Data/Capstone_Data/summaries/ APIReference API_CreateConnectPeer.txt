This documentation is about the AWS Global Networks for Transit Gateways API, specifically the CreateConnectPeer operation. This operation creates a core network Connect peer for a specified core network connect attachment between a core network and an appliance. The request syntax includes parameters such as BgpOptions, ClientToken, ConnectAttachmentId, CoreNetworkAddress, InsideCidrBlocks, PeerAddress, SubnetArn, and Tags. The response includes information about the ConnectPeer object. The documentation also provides information about possible errors and links to other related APIs and SDKs.

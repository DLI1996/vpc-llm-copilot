The AWS API Reference provides documentation for the RejectVpcPeeringConnection action in Amazon EC2. This action is used to reject a VPC peering connection request. The request parameters include the DryRun parameter, which checks for required permissions, and the VpcPeeringConnectionId parameter, which specifies the ID of the VPC peering connection. The response elements include the requestId and return values. The documentation also provides examples and links to language-specific AWS SDKs for using this API.

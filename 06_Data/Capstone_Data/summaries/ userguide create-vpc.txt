This document provides a user guide for creating a virtual private cloud (VPC) on Amazon Web Services (AWS). It explains the various configuration options available when creating a VPC, such as availability zones, CIDR blocks, DNS options, internet gateways, NAT gateways, route tables, subnets, and tenancy. The document also provides step-by-step instructions for creating a VPC and its associated resources using the AWS Management Console and the AWS CLI. Additionally, it includes information on tagging resources and creating a VPC with no additional resources.

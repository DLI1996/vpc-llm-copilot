This documentation provides step-by-step instructions on how to use open-source tools, such as Suricata and Zeek, to monitor network traffic from Amazon EC2 instances using Traffic Mirroring. It outlines the process of installing the Suricata software, creating a traffic mirror target, filter, and session. The example focuses on mirroring inbound TCP traffic to an instance with Suricata installed. The documentation also includes links to additional resources and provides options for monitoring mirrored traffic.

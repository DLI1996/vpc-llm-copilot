The AWS API Reference for Amazon EC2 provides documentation on the DescribeDhcpOptions action, which is used to retrieve information about one or more DHCP options sets. The action accepts parameters such as DhcpOptionsId, DryRun, Filter, MaxResults, and NextToken. The response includes elements such as dhcpOptionsSet, nextToken, and requestId. The documentation also includes examples and links to language-specific AWS SDKs for using the API.

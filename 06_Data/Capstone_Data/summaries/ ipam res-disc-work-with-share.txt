This document provides instructions on how to share a resource discovery using AWS Resource Access Manager (RAM) in the context of integrating an IP Address Manager (IPAM) with accounts outside of your organization. It outlines the steps to create a resource share, enable sharing with AWS Organizations, specify the principals allowed access to the shared resource, and associate the resource discovery with an IPAM. The document also mentions the required permissions and provides links to the AWS CLI Reference documentation for further details.

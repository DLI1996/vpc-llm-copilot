The AWS API Reference for Amazon EC2 provides documentation on the SearchTransitGatewayMulticastGroups action. This action allows users to search for one or more transit gateway multicast groups and retrieve information about their group membership. The action includes request parameters such as DryRun, Filter.N, MaxResults, NextToken, and TransitGatewayMulticastDomainId. The response elements include multicastGroups, nextToken, and requestId. The documentation also provides examples and links to language-specific AWS SDKs for further information.

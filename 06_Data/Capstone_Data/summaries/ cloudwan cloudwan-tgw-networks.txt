The AWS Cloud WAN User Guide provides documentation on AWS Network Manager, a service that allows users to manage and monitor their transit gateways. The guide covers various topics such as the overview of transit gateways, geography, topology tree, events, monitoring, and route analyzer. It provides step-by-step instructions on how to access and view information about transit gateways, VPN status, connect peer status, network events, and monitoring metrics. The guide also explains how to use CloudWatch Events and CloudWatch dashboards for tracking and analyzing transit gateway events and metrics.

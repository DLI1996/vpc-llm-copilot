The AWS CreateAccessLogSubscription API enables access logs to be sent to Amazon CloudWatch, Amazon S3, and Amazon Kinesis Data Firehose. The service network owner can use the access logs to audit the services in the network. The API request includes parameters such as clientToken, destinationArn, resourceIdentifier, and tags. The API response includes the ARN, destinationArn, id, resourceArn, and resourceId. Possible errors include AccessDeniedException, ConflictException, InternalServerException, ResourceNotFoundException, ThrottlingException, and ValidationException.

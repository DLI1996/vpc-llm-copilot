This documentation provides instructions on how to release a CIDR allocation from an IP Address Manager (IPAM) pool in Amazon VPC. It explains that you may need to release a pool allocation before deleting a pool, and provides steps for releasing an allocation in both private and public scopes. It also mentions that Amazon VPC IPAM will automatically release allocations after some time. The documentation includes AWS CLI commands for releasing a pool allocation and provides links to additional documentation for more detailed information.

This documentation is about the AWS API action "DeleteVpc", which is used to delete a specified VPC (Virtual Private Cloud). Before deleting the VPC, all associated gateways and resources must be detached or deleted. The request parameters include "DryRun" (to check permissions) and "VpcId" (the ID of the VPC). The response elements include "requestId" (the ID of the request) and "return" (a boolean indicating success or error). Common errors and examples are also provided. The documentation also mentions related APIs and language-specific SDKs for using the API.

This documentation provides information about the ConnectPeerAssociation API in AWS Global Networks for Transit Gateways. It describes the parameters and their constraints for creating a core network Connect peer association. The parameters include ConnectPeerId, DeviceId, GlobalNetworkId, LinkId, and State. The State parameter can have values like PENDING, AVAILABLE, DELETING, or DELETED. The documentation also provides links to language-specific AWS SDKs for more information on using this API.

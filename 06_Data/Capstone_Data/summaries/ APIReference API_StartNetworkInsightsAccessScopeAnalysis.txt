This documentation is about the AWS API action "StartNetworkInsightsAccessScopeAnalysis". It provides information about the request parameters, response elements, errors, and related resources. The action is used to start analyzing a specified Network Access Scope. The required parameters for this action include ClientToken and NetworkInsightsAccessScopeId. The response includes the Network Access Scope analysis and the ID of the request. The documentation also provides links to language-specific AWS SDKs for further information on using this API.

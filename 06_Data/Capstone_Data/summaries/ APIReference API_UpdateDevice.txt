This documentation is about the AWS Global Networks for Transit Gateways API, specifically the UpdateDevice operation. It provides information on the request syntax, URI request parameters, request body, response syntax, response elements, and possible errors. The UpdateDevice operation allows users to update the details of an existing device by specifying new values for parameters such as AWSLocation, Description, Location, Model, SerialNumber, SiteId, Type, and Vendor. The response includes information about the updated device.

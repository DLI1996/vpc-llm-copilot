This document provides a step-by-step guide on how to bring your own public IPv4 CIDR to IPAM (IP Address Manager) using only the AWS CLI. It covers creating AWS CLI named profiles and IAM roles, creating an IPAM, creating a top-level IPAM pool, provisioning a CIDR to the top-level pool, creating a regional pool within the top-level pool, provisioning a CIDR to the regional pool, sharing the regional pool, creating a public IPv4 pool, provisioning the public IPv4 CIDR to the public IPv4 pool, creating an Elastic IP address from the public IPv4 pool, advertising the CIDR, and performing cleanup steps.

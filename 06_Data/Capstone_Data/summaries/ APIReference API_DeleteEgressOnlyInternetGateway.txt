This is a summary of the documentation for the AWS API action "DeleteEgressOnlyInternetGateway". This action is used to delete an egress-only internet gateway. The request parameters include "DryRun" (a boolean value to check permissions) and "EgressOnlyInternetGatewayId" (the ID of the gateway to delete). The response elements include "requestId" (the ID of the request) and "returnCode" (a boolean indicating success or error). The documentation also provides examples and links to language-specific SDKs for using this API.

This documentation is for the AWS API action "DescribeNetworkInsightsAnalyses". It provides information about the parameters that can be used in the request, the response elements returned by the service, and any errors that may occur. The action is used to describe one or more network insights analyses. The parameters include analysis start and end times, a dry run option, filters, maximum results, network insights analysis IDs, and network insights path ID. The response includes information about the network insights analyses, a token for retrieving the next page of results, and the ID of the request.

This is a summary of the AWS Global Networks for Transit Gateways API reference documentation. The documentation describes the ConnectionHealth object, which provides information about the health of a connection. The object includes the connection status (UP or DOWN), the timestamp of the last status update, and the connection type (BGP or IPSEC). The documentation also provides links to language-specific AWS SDKs for further information on using the API. Additionally, it mentions that JavaScript must be enabled to access the AWS documentation.
